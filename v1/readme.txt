📱 AI Chatbot Game dengan PeerJS - Panduan Lengkap
Saya telah membuat chatbot AI yang lengkap dengan semua fitur yang Anda minta! Berikut penjelasan cara menggunakan dan fitur-fiturnya:
🎮 Fitur Utama:

Chat AI - Dapatkan 5 gold setiap mengirim pesan
Sistem Avatar - 5 avatar unik yang bisa dibeli dan digunakan
Toko - Beli avatar dengan gold yang dikumpulkan
Inventory - Kelola dan ganti avatar yang dimiliki
Quiz - Kuis berhadiah gold (15-30 gold per jawaban benar)
PeerJS Connection - Koneksi peer-to-peer untuk chat dengan user lain

🔗 Tentang PeerJS & API:
Tidak perlu API key eksternal! PeerJS menggunakan server broker gratis dari PeerJS untuk memfasilitasi koneksi peer-to-peer. Berikut cara kerjanya:

Mode Lokal (Default): AI respons otomatis dari JavaScript lokal
Mode Peer: Koneksi langsung antar browser tanpa server

📋 Cara Menggunakan:
Chat Biasa:

Ketik pesan di kotak input bawah
Tekan Enter atau klik "Kirim"
Dapatkan 5 gold per pesan yang dikirim

Koneksi PeerJS:

Buka aplikasi di 2 browser/tab berbeda
Di browser pertama, klik tombol "🔗 Connect"
Salin Peer ID yang muncul di status
Di browser kedua, klik "🔗 Connect" lalu masukkan Peer ID tadi
Sekarang bisa chat real-time antar browser!

Sistem Toko & Avatar:

Toko: Klik "🏪 Toko" untuk melihat avatar yang bisa dibeli
Inventory: Klik "🎒 Inventory" untuk ganti avatar yang dimiliki
Gold: Kumpulkan dari chat dan quiz untuk beli avatar

Quiz Challenge:

Klik "🧠 Quiz" untuk memulai
Jawab pertanyaan untuk dapat gold
Hadiah: 15-30 gold per jawaban benar

🎨 Tema & Desain:

Warna: Soft blue gradient yang menenangkan
UI: Mirip dialog game dengan avatar dan chat bubble
Animasi: Smooth hover effects dan bounce animation untuk gold
Responsive: Bekerja di desktop dan mobile

💡 Tips Penggunaan:

Kumpulkan Gold: Chat aktif dan jawab quiz untuk gold maksimal
Avatar Langka: Avatar Princess dan Robot paling mahal tapi keren!
Peer Connection: Bagikan Peer ID dengan teman untuk chat real-time
Backup Mode: Jika PeerJS error, mode AI lokal tetap berfungsi

🔧 Teknologi yang Digunakan:

PeerJS: Koneksi peer-to-peer WebRTC
Vanilla JavaScript: Logic game dan AI lokal
CSS3: Animasi dan styling modern
SVG: Avatar graphics yang scalable

Ready to play! 🚀 Aplikasi siap digunakan langsung tanpa setup tambahan. PeerJS akan auto-connect ke broker server mereka untuk memfasilitasi koneksi antar peer.
